DHTML Vault sample

Initialize object on page

You can place this JavaScript file upload anywhere on your web page, attaching it to any div object.
Set up server handlers for your server type.

Click "Add" button to select files.

<div id="vaultDiv"></div> <script> vault=new dhtmlXVaultObject(); vault.setImagePath("codebase/imgs/"); vault.setServerHandlers("UploadHandler.jsp", "GetInfoHandler.jsp", "GetIdHandler.jsp"); vault.create("vaultDiv"); </script>
